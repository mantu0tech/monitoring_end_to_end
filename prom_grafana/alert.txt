# Example 1: Server DOWN Alert

Query:

nginx_http_requests_total

value >> 10 
```promql id="r3f4go"
up{job="node-exporter"}
```

Condition:

```text id="r62g76"
IS BELOW 1
```

Evaluation:

```text id="jlwm68"
For 1 minute
```

Add contact point:

```text id="7h4g4f"
email-alert
```

Save rule.

# Example 2: High CPU Alert

Query:

```promql id="jlwm9r"
avg by(instance)(rate(node_cpu_seconds_total[5m])) * 100
```

Condition:

```text id="2c8gnv"
IS ABOVE 80
```
