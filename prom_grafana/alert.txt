# STEP 1: Configure SMTP in Grafana

Open Grafana config:

```bash id="0h4q5v"
sudo nano /etc/grafana/grafana.ini
```

Search:

```ini id="p0a8rz"
[smtp]
```

Replace with:

```ini id="btx81u"
[smtp]
enabled = true
host = smtp.gmail.com:587
user = YOUR_GMAIL@gmail.com
password = YOUR_APP_PASSWORD
from_address = YOUR_GMAIL@gmail.com
from_name = Grafana Alerts
skip_verify = true
```

# STEP 6: Create Alert Rule

```text id="7n75f8"
Alerts → Alert Rules
```

Click:

```text id="3bskf6"
New Alert Rule
```

---

# Example 1: Server DOWN Alert

Query:

```promql id="r3f4go"
up{job="node-exporter"}
```

Condition:

```text id="r62g76"
IS BELOW 1
```

Evaluation:

```text id="jlwm68"
For 1 minute
```

Add contact point:

```text id="7h4g4f"
email-alert
```

Save rule.

---

# Example 2: High CPU Alert

Query:

```promql id="jlwm9r"
100 - (avg by(instance)(rate(node_cpu_seconds_total{mode="idle"}[1m])) * 100)
```

Condition:

```text id="2c8gnv"
IS ABOVE 80
```

For:

```text id="7qtwix"
2 minutes
```

Save.

---

# Example 3: High Memory Alert

Query:

```promql id="p0f7xg"
(node_memory_MemTotal_bytes - node_memory_MemAvailable_bytes) / node_memory_MemTotal_bytes * 100
```

Condition:

```text id="y0pgo7"
IS ABOVE 90
```

Save.
